# Tushare 价格更新器使用说明

## ⚠️ 重要提示

**必须配置 Tushare token 才能使用本系统**

本系统从 **Tushare Pro** 获取真实的中国 A 股价格数据，不使用示例数据。

---

## 🚀 快速开始

### 步骤 1: 获取 Tushare Token

1. 访问 https://tushare.pro/
2. 注册免费账号
3. 登录后进入个人中心
4. 复制你的接口 Token

### 步骤 2: 配置 Token

#### 方法 1: 运行时输入（推荐）
```bash
python update_prices.py
# 程序会提示输入 token
```

#### 方法 2: 设置环境变量
```bash
# Windows
set TUSHARE_TOKEN=your_token_here

# Linux/Mac
export TUSHARE_TOKEN=your_token_here

# 然后运行
python update_prices.py
```

#### 方法 3: 在代码中设置
```python
from update_prices import RealTimePriceUpdater

updater = RealTimePriceUpdater(token='your_token_here')
```

## ⚠️ 常见问题

### Q1: 没有 token 会怎样？

系统会显示错误提示，无法获取数据。

**解决方法**: 必须注册获取 token

### Q2: Token 无效？

**原因**: 
- Token 复制错误
- Token 被重置

**解决方法**:
- 重新复制 token（包括所有字符）
- 在 tushare.pro 重新生成

### Q3: 调用次数超限？

**原因**: 基础用户每分钟 100 次调用

**解决方法**:
- 等待 1 分钟
- 减少调用频率
- 每日签到提高积分

### Q4: 数据已是最新？

**原因**: 当天已经更新过

**解决方法**: 
- 这是正常提示，无需操作
- 下一个交易日会自动更新

## 🔧 技术细节

### 数据源
- **Tushare Pro** - 中国 A 股专业数据源
- 数据准确、完整、及时

### 更新逻辑
1. 检查数据库最后更新日期
2. 如果需要更新，从 Tushare 获取新数据
3. 计算技术指标（MA, MACD, RSI, KDJ, BB, ATR, OBV）
4. 生成趋势信号
5. 保存到 SQLite 数据库

### 数据存储
- **daily_prices**: 价格数据（OHLCV）
- **technical_indicators**: 技术指标
- **trend_signals**: 趋势信号和推荐
