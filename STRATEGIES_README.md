# 市场策略增强版 - 使用说明

## 新增功能概述

本项目已增强添加了完整的市场交易策略系统，包含以下 6 种经典策略：

### 1. 趋势跟踪策略 (Trend Following)
- **原理**: 基于双均线交叉识别趋势
- **信号**: 
  - 金叉（短期均线上穿长期均线）→ 买入
  - 死叉（短期均线下穿长期均线）→ 卖出
- **参数**: 短期均线 5 日，长期均线 20 日
- **适用市场**: 趋势明显的牛市或熊市

### 2. 均值回归策略 (Mean Reversion)
- **原理**: 价格倾向于回归移动平均线
- **信号**:
  - 价格低于均线 2 倍标准差 → 买入（超卖）
  - 价格高于均线 2 倍标准差 → 卖出（超买）
- **参数**: 均线周期 20 日，标准差阈值 2.0
- **适用市场**: 震荡市

### 3. 动量策略 (Momentum)
- **原理**: 追涨杀跌，近期强势股票继续强势
- **信号**:
  - 20 日收益率 > 5% 且成交量放大 → 买入
  - 20 日收益率 < -5% 且成交量放大 → 卖出
- **参数**: 动量周期 20 日，信号阈值 5%
- **适用市场**: 趋势强劲的市场

### 4. 布林带策略 (Bollinger Bands)
- **原理**: 基于布林带的均值回归
- **信号**:
  - 价格触及下轨 → 买入
  - 价格触及上轨 → 卖出
- **参数**: 周期 20 日，标准差 2.0
- **适用市场**: 震荡市

### 5. MACD 交叉策略
- **原理**: 基于 MACD 指标的动量策略
- **信号**:
  - MACD 金叉 → 买入
  - MACD 死叉 → 卖出
- **参数**: 快线 12 日，慢线 26 日，信号线 9 日
- **适用市场**: 趋势市场

### 6. RSI 策略
- **原理**: 基于相对强弱指标的超买超卖
- **信号**:
  - RSI < 30 → 买入（超卖）
  - RSI > 70 → 卖出（超买）
- **参数**: RSI 周期 14 日
- **适用市场**: 震荡市

## 使用方法

### 快速开始

```bash
python main.py
```

### 运行流程

1. **输入股票代码** (如：000001 表示平安银行)

2. **选择分析模式**:
   ```
   1. ML Prediction (线性回归/随机森林预测)
   2. Market Strategy Analysis (单个策略分析)
   3. Compare All Strategies (比较所有策略)
   4. Exit (退出)
   ```

3. **策略分析模式**:
   - 选择单个策略进行详细分析
   - 或选择运行所有策略比较

4. **查看结果**:
   - 策略收益率
   - 胜率
   - 夏普比率
   - 最大回撤

### 使用策略分析器 API

```python
from strategy_analyzer import StockStrategyAnalyzer

# 创建分析器
analyzer = StockStrategyAnalyzer(initial_cash=1000000)

# 分析单个策略
result = analyzer.analyze_single_strategy('trend_following', stock_data)

# 比较所有策略
comparison = analyzer.compare_all_strategies(stock_data)

# 生成报告
report = analyzer.generate_strategy_report(stock_data)
```

## 策略回测引擎

### 功能特点

- **初始资金**: 默认 100 万
- **交易成本**: 佣金万分之三
- **滑点**: 0.1%
- **性能指标**:
  - 总收益率
  - 胜率
  - 夏普比率
  - 最大回撤
  - 平均收益

### 性能指标说明

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 总收益率 | 策略总收益百分比 | > 20% |
| 胜率 | 盈利交易占比 | > 55% |
| 夏普比率 | 风险调整后收益 | > 1.0 |
| 最大回撤 | 最大亏损幅度 | < 20% |

## 策略选择建议

### 不同市场环境

| 市场环境 | 推荐策略 | 原因 |
|----------|----------|------|
| 牛市 | 趋势跟踪、动量策略 | 顺势而为，获取最大收益 |
| 震荡市 | 均值回归、布林带、RSI | 高抛低吸 |
| 熊市 | 空仓观望 | 保存实力 |

### 风险偏好

| 风险偏好 | 推荐策略 |
|----------|----------|
| 保守型 | 均值回归、RSI |
| 平衡型 | 趋势跟踪、MACD |
| 进取型 | 动量策略、布林带突破 |

## 风险提示

1. **历史回测不代表未来表现**: 过去的收益不能保证未来同样盈利
2. **过拟合风险**: 策略可能在历史数据上表现好，但实盘效果差
3. **交易成本**: 实际交易中存在佣金、印花税、滑点等成本
4. **市场变化**: 市场条件变化可能导致策略失效
5. **资金管理**: 建议分散投资，不要全仓单一策略

## 常见问题

### Q: 策略回测准确吗？
A: 回测基于历史数据，已考虑交易成本和滑点，但实际交易仍可能有差异。

### Q: 如何选择最优策略？
A: 建议使用 `compare_all_strategies` 比较所有策略，选择夏普比率高、回撤小的策略。

### Q: 可以实盘交易吗？
A: 本系统仅供学习和研究，实盘交易需谨慎，建议先模拟盘测试。
