# 股票推荐功能 - 使用说明

## 📌 功能概述

股票推荐功能基于 6 种市场策略的综合评分，为用户提供买卖建议和风险提示。

### 核心特性

- **多策略信号综合**: 整合趋势跟踪、均值回归、动量、布林带、MACD、RSI 六种策略
- **综合评分系统**: -10 到 10 的评分范围，直观显示股票吸引力
- **推荐等级**: STRONG_BUY, BUY, HOLD, SELL, STRONG_SELL
- **目标价和止损**: 自动计算建议的目标价格和止损价格
- **风险评估**: LOW, MEDIUM, HIGH 三级风险评估
- **详细原因**: 列出每个策略的推荐原因

## 🚀 使用方法

### 方法 1: 独立运行推荐系统

```bash
python run_stock_recommender.py
```

**功能选项**:
1. 使用示例数据演示 (快速测试)
2. 分析单只股票
3. 分析股票池 (多只股票)
4. 自定义股票列表

### 方法 2: 在主程序中使用

```bash
python main.py
# 输入股票代码
# 选择 4: Get Stock Recommendation
```

### 方法 3: 使用 API

```python
from stock_recommender import StockRecommender

# 创建推荐引擎
recommender = StockRecommender()

# 分析单只股票
signal = recommender.analyze_stock(stock_data, '000001', '平安银行')

# 获取推荐
print(f"推荐等级：{signal.recommendation}")
print(f"综合评分：{signal.score:.2f}")
print(f"目标价格：{signal.target_price}")
print(f"止损价格：{signal.stop_loss}")

# 分析多只股票
recommender.analyze_multiple_stocks(stock_data_dict)

# 获取买入推荐
buy_recs = recommender.get_buy_recommendations(top_n=5)
for signal in buy_recs:
    print(f"{signal.symbol}: {signal.recommendation} (评分：{signal.score:.2f})")
```

## 📊 推荐系统详解

### 评分系统

| 评分范围 | 推荐等级 | 含义 |
|----------|----------|------|
| 7 ~ 10 | STRONG_BUY | 强烈买入 |
| 4 ~ 7 | BUY | 买入 |
| -4 ~ 4 | HOLD | 持有/观望 |
| -7 ~ -4 | SELL | 卖出 |
| -10 ~ -7 | STRONG_SELL | 强烈卖出 |

### 策略权重

| 策略 | 权重 | 说明 |
|------|------|------|
| 趋势跟踪 | 1.5 | 最高权重，趋势是核心 |
| 动量策略 | 1.3 | 重视 momentum 效应 |
| MACD | 1.2 | 经典指标 |
| 布林带 | 1.0 | 标准权重 |
| 均值回归 | 1.0 | 标准权重 |
| RSI | 1.0 | 标准权重 |

### 信号说明

| 信号值 | 含义 | 说明 |
|--------|------|------|
| 2 | 强烈买入 | 策略发出强烈买入信号 |
| 1 | 买入 | 策略发出买入信号 |
| 0 | 持有 | 策略无明确方向 |
| -1 | 卖出 | 策略发出卖出信号 |
| -2 | 强烈卖出 | 策略发出强烈卖出信号 |

## 🔧 技术细节

### 目标价和止损计算

**买入信号**:
- 目标价 = 当前价 + 2 × ATR (或 +10%)
- 止损价 = 当前价 - 1.5 × ATR (或 -5%)

**卖出信号**:
- 目标价 = 当前价 - 2 × ATR (或 -10%)
- 止损价 = 当前价 + 1.5 × ATR (或 +5%)

其中 ATR (Average True Range) 为平均真实波幅，衡量股票波动性。

### 风险评估

| 等级 | 波动率 | 最大回撤 |
|------|--------|----------|
| LOW | < 30% | < 15% |
| MEDIUM | 30% - 50% | 15% - 25% |
| HIGH | > 50% | > 25% |
