## 🚀 核心功能

### 1. Web 可视化仪表盘 (dashboard.py)

**功能模块**:
- 🏠 **首页** - 功能卡片、最新推荐、使用流程
- 🔍 **个股分析** - K 线图、策略信号、推荐分析
- 📊 **股票推荐** - 推荐列表、等级分布
- 🎯 **股票筛选** - 多维度条件筛选
- 📈 **策略回测** - 策略收益对比
- 💼 **投资组合** - 持仓跟踪、盈亏计算

**技术特性**:
- Plotly 交互式图表
- Streamlit 响应式布局
- 实时数据更新
- 深色主题支持

### 2. 预警监控面板 (alert_dashboard.py)

**功能模块**:
- 📋 **预警列表** - 活跃/已触发/全部预警
- ➕ **设置预警** - 5 种预警类型
- ⚙️ **通知配置** - 邮件/微信/短信
- 📊 **监控状态** - 统计数据

**预警类型**:
1. **价格突破预警** - 向上/向下突破
2. **止损止盈预警** - 自动计算价格
3. **RSI 超买超卖** - 技术指标预警
4. **成交量突破** - 成交量异常
5. **均线交叉** - 技术信号

### 3. 实时监控引擎 (realtime_monitor.py)

**核心类**:
- `RealtimeMonitor` - 完整监控器（需要 Tushare token）
- `SimpleMonitor` - 简易监控器（无需 token）

**功能**:
- 实时价格监控
- 自动预警检查
- 多线程支持
- 可配置检查间隔

### 4. 价格预警模块 (price_alert.py)

**核心类**:
- `PriceAlert` - 预警配置对象
- `PriceAlertManager` - 预警管理器
- `AlertType` - 预警类型枚举
- `AlertStatus` - 预警状态枚举

**功能**:
- 创建/取消/删除预警
- 预警条件检查
- 自动过期管理
- JSON 持久化存储

### 5. 通知服务 (notification_service.py)

**支持方式**:
- 📧 **邮件通知** - SMTP 协议
- 💬 **微信通知** - ServerChan 推送
- 📱 **短信通知** - 阿里云短信（可扩展）
- 💻 **控制台通知** - 开发测试

**功能**:
- 多通知方式支持
- HTML 邮件模板
- 配置管理
- 测试功能

## 💻 使用示例

### 启动 Web 仪表盘

```bash
# 安装依赖
pip install -r requirements_web.txt

# 启动股票分析仪表盘
streamlit run web_app/dashboard.py

# 启动预警监控面板
streamlit run web_app/alert_dashboard.py
```

**访问地址**: http://localhost:8501

### 设置价格预警

```python
from price_alert import PriceAlertManager

manager = PriceAlertManager()

# 创建价格突破预警
manager.create_price_breakout_alert(
    symbol="000001",
    name="平安银行",
    breakout_price=10.5,
    above=True
)

# 创建止损预警
manager.create_stop_loss_alert(
    symbol="000001",
    name="平安银行",
    stop_price=9.5
)
```

### 实时监控

```python
from realtime_monitor import SimpleMonitor

monitor = SimpleMonitor()

# 添加监控股票
monitor.add_stock("000001", "平安银行")

# 设置止损止盈
monitor.set_stop_loss_take_profit(
    "000001", "平安银行",
    current_price=10.0,
    stop_loss_pct=5.0,
    take_profit_pct=10.0
)

# 手动检查
monitor.check_manual("000001", current_price=10.6)
```

### 配置邮件通知

```python
from notification_service import NotificationService

service = NotificationService()

service.setup_email(
    smtp_server="smtp.qq.com",
    smtp_port=587,
    sender="your_email@qq.com",
    password="your_auth_code",
    receivers=["receiver@email.com"]
)
```

## 🎯 使用场景

### 场景 1: 上班族盯盘

**问题**: 工作忙，无法实时看盘

**解决方案**:
1. 设置价格预警（突破关键位）
2. 设置止损止盈预警

4. 触发预警自动通知

**问题**: 等待技术突破，容易错过

1. 设置均线交叉预警
2. 设置布林带突破预警
3. 设置 RSI 超买超卖预警
4. 信号出现立即通知

## 🔧 配置说明

### 配置文件位置

```
config/config.json
```

### 邮件通知配置

```json
{
  "email": {
    "enabled": true,
    "smtp_server": "smtp.qq.com",
    "smtp_port": 587,
    "sender": "your_email@qq.com",
    "password": "your_auth_code",
    "receivers": ["receiver@email.com"]
  }
}
```

### 微信通知配置

1. 访问 https://sct.ftqq.com/
2. 获取 ServerChan 推送 key
3. 配置:

```json
{
  "wechat": {
    "enabled": true,
    "method": "serverchan",
    "serverchan_key": "SCTxxxxxxxxxxxxxxxx"
  }
}
```

## ❓ 常见问题

### Q1: Streamlit 无法启动？

**A**: 检查依赖是否安装:
```bash
pip install -r requirements_web.txt
```

### Q2: 如何修改端口？

**A**: 
```bash
streamlit run web_app/dashboard.py --server.port 8502
```

### Q3: 预警不触发？

**A**:
1. 检查预警状态是否为 active
2. 确认价格是否达到触发条件
3. 检查预警是否过期

### Q4: 邮件发送失败？

**A**:
1. 使用授权码而非登录密码
2. 确认 SMTP 服务已开启
3. 检查防火墙设置

## 🎓 最佳实践

### 1. 合理设置预警位

- **止损位**: 建议 5-8%
- **止盈位**: 建议 10-20%
- **突破位**: 参考技术阻力/支撑位

### 2. 选择通知方式

| 方式 | 适用场景 | 实时性 |
|------|---------|--------|
| 控制台 | 开发测试 | 即时 |
| 邮件 | 正式使用 | 较准时 |
| 微信 | 手机接收 | 即时 |

### 3. 监控频率设置

- **短线交易**: 30-60 秒
- **中线交易**: 5-10 分钟
- **长线投资**: 1 小时

### 4. 定期清理

- 每周清理过期预警
- 查看已触发记录
- 调整不合理预警位
