## 已知问题

### 1. Yahoo Finance 速率限制

**问题描述：** 频繁请求时会出现 `YFRateLimitError`

**影响范围：** 
- 实时价格获取
- 投资组合盈亏计算
- 预警检查

**解决方案：**
1. 配置 Tushare Token（推荐）
2. 增加请求间隔
3. 使用数据缓存

### 2. Streamlit 安装

**问题描述：** Streamlit 及其依赖包较大，安装时间较长

**解决方案：** 使用预编译二进制包安装
```bash
pip install streamlit --only-binary :all:
```
